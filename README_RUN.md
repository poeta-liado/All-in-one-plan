# Run the NexusCRM prototype (minimal scaffold)

1. Install dependencies:

```bash
cd /workspaces/All-in-one-plan
npm install
```

2. Start the server:

```bash
npm start
```

3. Open the prototype in your browser:

- `nexuscrm.html` → http://localhost:3000/nexuscrm.html
- `nexuscrm (2).html` → http://localhost:3000/nexuscrm%20(2).html

Notes:
- This scaffold uses a simple Express static server and lightweight API stubs. It does not configure a database or Prisma.
